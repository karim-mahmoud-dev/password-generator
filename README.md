# 🔑 Password Generator

A clean, high-performance random password generator built as a Scrimba Solo Project.

## 🚀 Features
- **Instant Generation:** Generates two unique, secure passwords simultaneously.
- **Modern UI:** Dark-themed interface with high-contrast green accents.
- **Interactive Design:** Hover effects on action buttons and output fields.
- **Responsive:** Optimized for different screen sizes.

## 🛠️ Tech Stack
- **HTML5** - **CSS3** (Flexbox & Custom Transitions)
- **JavaScript** (ES6+ Logic & DOM Manipulation)

## 💡 What I Learned
During this project, I focused on:
- Managing application state with JavaScript variables to handle real-time UI updates.
- Implementing secure random logic using `Math.random()`.
- Refined my CSS skills by using Flexbox for precise alignment of each element.
